# ГЭС кофе-бот

### Установка
```pip3 install -r requirements.txt```

- В файле ```config.py``` прописать telegram-токен и chat-id бариста.
- В файле ```messages.py``` заполнить списки напитков и мест.

### Запуск
```python3 main.py```

### Доступные команды

- ```\id``` - узнать chat-id (для бариста).
- ```\start``` - начать заказ напитков.
